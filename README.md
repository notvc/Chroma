# Chroma — Professional Color Studio 🎨

Chroma is a powerful, professional-grade color picker and gradient builder built with **pure vanilla JavaScript, HTML, and CSS**. No frameworks, no bloat—just a clean, modular, and performant tool for developers and designers.

## 🚀 Features

- **Advanced Color Picker**: Fine-tune your colors with saturation, brightness, hue, and opacity controls on an interactive canvas.
- **Multiple Color Formats**: Instant conversion and quick-copy for `HEX`, `RGB`, `RGBA`, `HSL`, and `HSLA`.
- **WCAG Contrast Checker**: Automatically check contrast ratios against white and black backgrounds for accessibility (AA/AAA grading).
- **Gradient Builder**: Create complex linear and radial gradients, manage color stops, and generate CSS code instantly.
- **Color History**: Automatically tracks your recently used colors (up to 20).
- **Favorites & Palettes**: Save your favorite colors. Export and import your color palettes as JSON files to share or backup.
- **Eyedropper Tool**: Pick colors directly from your screen (supports browsers with the native EyeDropper API).
- **Theme Support**: Seamlessly toggle between Dark and Light modes.
- **Local Storage**: Your history, favorites, and theme preferences are saved locally and persist between sessions.

## ⌨️ Keyboard Shortcuts

Speed up your workflow with built-in shortcuts:

- <kbd>C</kbd> - Copy current HEX value
- <kbd>R</kbd> - Generate random color
- <kbd>E</kbd> - Activate Eyedropper
- <kbd>T</kbd> - Toggle Dark/Light theme
- <kbd>S</kbd> - Save current color to favorites

## 🛠️ Tech Stack

- **HTML5**: Semantic markup with inline SVG icons.
- **CSS3**: Custom styles, responsive layouts, and CSS variables for theming.
- **JavaScript (ES6+)**: Pure JS, canvas manipulation, and advanced color math algorithms (`script.js`).

## 📦 Getting Started

Since Chroma is built without any build tools or dependencies, getting started is incredibly simple!

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/chroma.git
   ```
2. Open `index.html` in your favorite modern browser.
3. Start picking colors!
